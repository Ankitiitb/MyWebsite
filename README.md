# Ankit Sharma - Academic Projects Portfolio

A professional portfolio website showcasing academic projects from Semesters 7-10 in Aerospace Engineering.

## 📋 Project Overview

This portfolio highlights diverse aerospace engineering projects including:
- **Finite Element Method (FEM)** - Structural analysis and computation
- **Numerical Methods** - Advanced computational techniques
- **Aeroacoustics** - Sound generation in aerodynamic flows
- **Interfacial Phenomena** - Multi-phase flow analysis
- **Particle Methods** - Computational particle dynamics
- **Thermoacoustics** - Thermal-acoustic interactions
- **Turbulence** - Turbulent flow analysis
- **Aeroelasticity** - Fluid-structure interaction
- **Droplet Dynamics Project (DDP)** - Advanced research on droplet breakup

## 🚀 How to Deploy on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and create a new repository
2. Name it: `yourusername.github.io` (replace `yourusername` with your GitHub username)
3. Make it public

### Step 2: Upload Your Website Files
1. Clone the repository to your computer:
   ```bash
   git clone https://github.com/yourusername/yourusername.github.io.git
   cd yourusername.github.io
   ```

2. Copy all files from the `My_website` folder to the cloned repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md` (this file)

### Step 3: Push to GitHub
1. Navigate to your repository folder:
   ```bash
   cd yourusername.github.io
   ```

2. Stage and commit your files:
   ```bash
   git add .
   git commit -m "Initial portfolio website"
   git push origin main
   ```

### Step 4: Access Your Portfolio
Your website will be live at: `https://yourusername.github.io`

It may take a few minutes to deploy.

## 🎨 Customization

You can easily customize the portfolio:

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c3e50;      /* Main color */
    --secondary-color: #3498db;    /* Accent color */
    --accent-color: #e74c3c;       /* Highlight color */
}
```

### Add Project Details
Edit project descriptions in `index.html` in the project card sections.

### Update Personal Information
Modify the name, tagline, and other personal details in `index.html`.
